## คำอธิบายยาว (Long Description — สูงสุด 4000 ตัวอักษร)

🍼 DreamBook — สมุดบันทึกลูกน้อย
แอปติดตามการให้นม ผ้าอ้อม การนอน และพัฒนาการลูกน้อย ที่เน้นความเป็นส่วนตัวอันดับหนึ่ง

ไม่ต้องสมัครบัญชี ไม่ต้องล็อกอิน ข้อมูลไม่ออกจากโทรศัพท์คุณโดยไม่ได้รับอนุญาต แค่คุณ ครอบครัว และทุกช่วงเวลาของลูก — เก็บไว้อย่างปลอดภัยด้วยการเข้ารหัสระดับธนาคาร

━━━━━━━━━━━━━━━━━━━━━━━
🔒 ความเป็นส่วนตัวมาก่อน — ข้อมูลของคุณ กฎของคุณ
━━━━━━━━━━━━━━━━━━━━━━━

แอปบันทึกลูกทั่วไปส่งข้อมูลไปคลาวด์ DreamBook ไม่ทำแบบนั้น

• ทุกอย่างถูกเข้ารหัสในโทรศัพท์ทันทีที่คุณพิมพ์
• การแชร์กับผู้ดูแลใช้การเข้ารหัสตลอดเส้นทาง — แม้แต่เราก็อ่านไม่ได้
• ไม่ต้องสมัครบัญชี ไม่ต้องวันนี้ ไม่ต้องวันไหนทั้งนั้น
• ไม่มีโฆษณา ไม่มีตัวติดตาม ไม่มี SDK วิเคราะห์พฤติกรรม
• ข้อมูลสุขภาพลูกเป็นของครอบครัวคุณเท่านั้น

━━━━━━━━━━━━━━━━━━━━━━━
🍼 ติดตามการให้นม
━━━━━━━━━━━━━━━━━━━━━━━

• นาฬิกาจับเวลาแยกซ้าย-ขวา กดเริ่ม กดสลับ ง่ายมาก
• บันทึกนมขวดเป็น oz หรือ ml
• บันทึกการปั๊มนม (แยกซ้าย-ขวา ปริมาณต่อครั้ง)
• ติดตามสต็อกนมแช่แข็งพร้อมแจ้งเตือนวันหมดอายุ — ไม่มีถุงไหนสูญเปล่า
• ดูประวัติการให้นมและยอดรวมรายวันได้ทันที

━━━━━━━━━━━━━━━━━━━━━━━
💤 ติดตามการนอน
━━━━━━━━━━━━━━━━━━━━━━━

• กดเริ่มเมื่อลูกหลับ กดหยุดเมื่อลูกตื่น แค่นั้น
• แยกงีบกลางวันและนอนกลางคืน
• ดูชั่วโมงการนอนรายวันและรายสัปดาห์
• ดูรูปแบบการนอนย้อนหลัง

━━━━━━━━━━━━━━━━━━━━━━━
🧷 บันทึกผ้าอ้อม
━━━━━━━━━━━━━━━━━━━━━━━

• บันทึกเร็ว — เปียก สกปรก หรือทั้งคู่
• เพิ่มโน้ตสำหรับสิ่งผิดปกติที่สังเกตได้
• ยอดรวมผ้าอ้อมรายวันอยู่ในสรุปประจำวัน

━━━━━━━━━━━━━━━━━━━━━━━
💉 บันทึกวัคซีน
━━━━━━━━━━━━━━━━━━━━━━━

• บันทึกวัคซีนพร้อมวันที่ เข็มที่ และโน้ต
• ไม่ลืมว่าลูกได้รับอะไรไปแล้วในครั้งก่อน
• ไปหมอทีไรมั่นใจทุกที

━━━━━━━━━━━━━━━━━━━━━━━
📋 สรุปประจำวัน
━━━━━━━━━━━━━━━━━━━━━━━

ทุกวันรวมให้อัตโนมัติ — ยอดนม ออนซ์ ผ้าอ้อม และชั่วโมงการนอน เอาไปรายงานกุมารแพทย์ได้เลย หรือแค่ใช้ให้รู้สึกว่าตัวเองยังจัดการได้ในช่วงทารกแรกเกิด

━━━━━━━━━━━━━━━━━━━━━━━
👨‍👩‍👧 แชร์กับผู้ดูแล (Premium)
━━━━━━━━━━━━━━━━━━━━━━━

• แชร์กับแฟน ปู่ย่าตายาย หรือพี่เลี้ยง ด้วยรหัสเชิญ 8 ตัวอักษร
• ซิงค์แบบเรียลไทม์ — รู้ทันทีว่าให้นมครั้งสุดท้ายเมื่อไหร่
• เข้ารหัสตลอดเส้นทาง: เป็นส่วนตัวโดยการออกแบบ ไม่ใช่แค่นโยบาย
• ไม่มีโปรไฟล์สาธารณะ ไม่มีโซเชียล — แชร์ในครอบครัวเท่านั้น

แฝด? ลูกคนที่สอง? สลับระหว่างลูกได้ทันที บันทึกแยกกันชัดเจน

• สำรองข้อมูลขึ้นคลาวด์ — ยังเข้ารหัสตลอดเส้นทางอยู่

• วลีกู้คืน BIP-39: รหัส 12 คำ (เหมือน hardware wallet คริปโต) ให้คุณควบคุมข้อมูลได้เสมอ
