# DreamBook — Release Notes v1.0

## English (500 chars max)

Welcome to DreamBook!

Track every feed (breast + bottle), diaper, nap, pump session, and vaccine — all in one place.

Your baby's data is encrypted on your device. No account needed, ever.

Premium: real-time caregiver sharing, multi-baby support, Visit Summary PDF, and encrypted cloud backup with BIP-39 recovery.

7-day free trial. Thank you for trusting us with what matters most.

## ภาษาไทย (500 ตัวอักษร)

ยินดีต้อนรับสู่ DreamBook!

บันทึกการให้นม (แม่ + ขวด) ผ้าอ้อม การนอน การปั๊ม และวัคซีน ครบในที่เดียว

ข้อมูลลูกน้อยเข้ารหัสอยู่ในโทรศัพท์คุณ ไม่ต้องสมัครบัญชีเลย

พรีเมียม: แชร์กับผู้ดูแลแบบเรียลไทม์ รองรับหลายคน ส่งออก PDF สำหรับหมอ และสำรองข้อมูลบนคลาวด์แบบเข้ารหัสพร้อมวลีกู้คืน BIP-39

ทดลองฟรี 7 วัน ขอบคุณที่ไว้วางใจเรากับสิ่งที่สำคัญที่สุด
